# MigraCast – Datasets

## 1. Migraine Dataset from Wearable Devices

Purpose:
Main training dataset.

Main label:
`migraine_occurrence`

Features:
- sleep_hours
- mood_level
- stress_level
- hydration_level
- screen_time
- migraine_severity

Usage:
- supervised learning
- migraine prediction
- baseline model

---

## 2. Sleep Health and Lifestyle Dataset

Purpose:
Feature expansion.

Features:
- Sleep Duration
- Quality of Sleep
- Physical Activity Level
- Stress Level
- Heart Rate
- Daily Steps

Usage:
- additional lifestyle features
- feature validation
- correlation analysis

---

## 3. Open-Meteo API

Purpose:
External environmental features.

Features:
- temperature
- humidity
- air pressure
- precipitation

Usage:
- real-time weather integration
- environmental trigger analysis

---

## Dataset Strategy

The project combines:
- labeled migraine data
- lifestyle data
- environmental data

to build a multimodal prediction system.

---

## Important Notes

Some datasets are partially synthetic or generalized.

The project therefore focuses on:
- experimentation
- architecture
- ML workflow
- personalization concepts
rather than medical accuracy.
